Unigram

Complemento que agrega algunos comandos para simplificar el uso de esta aplicación.


Comandos:

  • control + c; este comando sirve tanto para enfocar la lista de chats como para volver a la lista de mensajes cuando hay una conversación abierta. Para ir desde la lista de mensajes a la de chats, debemos cerrar la conversación con escape 2 veces, y luego el atajo control + c.
  • control + barra espaciadora; reproduce y pausa los mensajes de voz desde la lista de mensajes.
  • control + d; activa y desactiva la velocidad doble en los mensajes de voz en reproducción..
  • control + l; si el mensaje tiene un link, lo abre en el navegador por defecto."""
